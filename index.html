Python code for generating strict partitions with parts at most max

25 Jul 2015 

def partition(n,max):
    if not n:
        yield ()
    for i in range(min(n,max),0,-1):
        for parts in partition(n-i,min(max,i-1)):
            yield (i,)+parts

This is so trivial or uncared-for that I wasn't able to copy-paste from either stackoverflow or activestate. If it turns out to be too slow for your purposes, try modifying Jerome Keller's code for ascending compositions.
